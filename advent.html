Advent Of Code Challenges

Advent of Code is a collection of coding puzzles that has multiple different levels of difficulty. These are able to be solved by everyone, and are released daily during the month of December. Since it was released in December, all of the puzzles were Christmas themed.

These puzzles helped me improve my knowledge of JavaScript, since I started doing these puzzles after just beginning to learn the language. For instance, one of the first puzzles taught me what and how to utilize the map function, which I have utilized multiple times ever since I learned it.

For me, these puzzles were almost as important as just learning the basics of JavaScript, because it taught me how to think about solving problems, rather than just simply getting from Point A to Point B.
